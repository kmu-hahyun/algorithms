# 공간 복잡도
- 512MB
  -	int는 1억 개가 512MB 기준 대략 한계
  -	bool은 5억 개가 512MB 기준 대략 한계
  - O(n^2) n의 한계 ≈ 11,500

# 시간 복잡도
- 1억/초 기준

## 시간 복잡도 빠른 순서
- $O(1)$
- $O(\log n)$
- $O(n)$
- $O(n \log n)$ - n <= 10^5 ~ 10^6(상수항이 작을 경우)
- $O(n^{2})$ - n <= 3000 ~ 5000
- $O(n^{3})$ - n <= 300 ~ 500
- $O(2^{n})$ - n <= 20 ~ 25
- $O(n!)$
